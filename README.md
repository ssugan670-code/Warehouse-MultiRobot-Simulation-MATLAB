# Warehouse Multi-Robot Simulation using MATLAB

This repository contains an autonomous multi-agent fleet coordination and routing framework developed for the MathWorks Excellence in Innovation Program (Project Challenge 212). 

## Project Objective
To design and simulate multi-robot operations inside a dynamic warehouse layout, focusing on:
- Multi-Agent Path Planning (Extended A* Algorithm)
- Task Scheduling & Fleet Allocation
- Real-time Cross-Collision Avoidance Logic

## Toolboxes Used
- Robotics System Toolbox™
- Navigation Toolbox™
